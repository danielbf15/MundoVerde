# Mundo Verde – Prototipo básico con navegación de contenidos, quizzes y panel docente. Guarda progreso localmente.

**Mundo Verde** es un prototipo funcional diseñado para apoyar el aprendizaje en instituciones educativas rurales con limitaciones de conectividad a Internet. Este proyecto propone una plataforma offline que permite a docentes y estudiantes acceder a recursos educativos, registrar avances y gestionar información académica sin requerir conexión externa, garantizando continuidad educativa en contextos con infraestructura limitada.



## Objetivo del Proyecto

Desarrollar una plataforma educativa offline, de fácil acceso y bajo consumo de recursos, que contribuya a disminuir la brecha digital en comunidades rurales mediante herramientas tecnológicas adaptadas a su contexto.



## Funcionalidades del Prototipo

- **Acceso offline total**: Los contenidos y recursos funcionan sin conexión a Internet.  
- **Modelo de datos local**: Gestión de información educativa mediante almacenamiento local.  
- **Interfaz amigable**: Navegación simple para estudiantes y docentes.  
- **Módulos del prototipo**:  
  - Gestión de contenidos educativos.  
  - Registro básico de progreso académico.  
  - Visualización de recursos y actividades.  
- **Arquitectura ligera**: Pensada para equipos de bajo rendimiento.
- **Carga de lecciones**: A través de JSON siguiendo la estructura de `plantilla.json`.



## Tecnologías Implementadas

- **HTML5**, **CSS3**, **JavaScript**
- **JSON** para almacenamiento y estructura de datos  
- **Arquitectura modular**  
- **Archivos estáticos** optimizados para uso offline  
- **Buenas prácticas** en diseño responsivo y accesibilidad  

Estas tecnologías fueron elegidas por su capacidad de funcionar en condiciones de baja capacidad computacional y por su naturaleza abierta, sostenible y portable.
